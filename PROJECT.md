# 月間昼食アドバイザー プロジェクト

## 目的

つくかね麻布の月間献立とセブン-イレブン候補を比較し、在宅勤務時の午後の眠気を考慮した昼食のおすすめを、PCとスマホから確認できるようにする。

## 現在の完成状態

- 2026年7月の月間献立を登録済み
- PC向けローカル管理画面を用意済み
- スマホ対応の公開ページをGitHub Pagesで公開済み
- 公開URLは固定：<https://imataka1995-star.github.io/lunch-advisor/index.html>
- 毎日の自動実行は行わず、月ごとの更新運用

## 使い始める場所

| やりたいこと | 開くもの |
| --- | --- |
| 毎日のおすすめを確認 | 公開URLをブラウザで開く |
| 翌月の献立を登録 | PCで `lunchadvisor://admin` を開く |
| 現在のローカルHTMLを確認 | `reports\\current.html` |
| 技術的な再開・更新 | [HANDOFF.md](HANDOFF.md) |

## 毎月の定型作業

1. PCの管理画面で翌月の献立画像を登録する。
2. A/B候補・カロリー・副菜・おすすめ理由を確認する。
3. Codexに「昼食アドバイザーの公開ページを更新して」と依頼する。
4. 同じ公開URLをPCとスマホで開き、新しい月になっていることを確認する。

公開用HTMLは `publish\\index.html`、公開先は GitHub の `imataka1995-star/lunch-advisor` リポジトリである。秘密情報やAPIキーは公開先に置かない。

## 引き継ぎ

設定内容、公開方法、確認済みの動作、注意点は [HANDOFF.md](HANDOFF.md) に記録済み。次の担当者または次回のCodexには、このファイルと `HANDOFF.md` を最初に読ませる。
