# lunch-advisor

つくかね弁当の月間献立と、セブン-イレブンの固定候補を比較する昼食アドバイザーです。

## 公開ページ

https://imataka1995-star.github.io/lunch-advisor/index.html

PCでは一覧表、スマートフォンでは日ごとのカード表示になります。上部の「今日」「月間ベスト」「第1週〜第5週」から目的の場所へ移動できます。

## 毎月の更新

1. PCの献立登録画面から翌月の献立画像を登録します。
2. 月間レポートを生成し、公開用の `publish/index.html` を更新します。
3. Codexに「昼食アドバイザーの公開ページを更新して」と依頼します。
4. このリポジトリの `index.html` が差し替わり、同じ公開URLで新しい月が表示されます。

公開リポジトリには閲覧用HTMLだけを配置し、APIキーなどの秘密情報は保存しません。
